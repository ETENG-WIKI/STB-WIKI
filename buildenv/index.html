前台开发环境搭建

建议以下软件安装最新版

  1. 安装nodejs

    下载地址：https://nodejs.org/en/

  2. 安装git(版本控制工具)

    下载地址：https://git-scm.com/

  3. 安装bowe(前台包管理器)

    安装方法：

    windows:npm install -g bower
linux:  sudo npm install -g bower
    1. 安装grunt-cli(构建编译工具)

    安装方法:

    windows: npm install -g grunt-cli
linux:  sudo npm install -g grunt-cli
